总结

ConcurrentHashMap使用到的技术点：

  1. CAS+自旋 乐观锁的思想减少线程上下文切换
  2. 分段锁的思想 减少同一把锁争用带来的低效问题
  3. CounterCell 分段存储元素个数 减少多线程同时更新一个字段带来的低效
  4. @sun.misc.Contended(CounterCell上的注解) 避免伪共享
  5. 多线程协同进行扩容
