WeakHashMap

发表于 2019-08-08 更新于 2019-08-10 分类于 Java集合

WeakHashMap是一种弱引用map，内部key会存储为弱引用，当JVM GC时 若这些key没有强引用存在时就会被gc回收，下次当我们操作map时会将对应的
Entry这个删除掉，基于这个特性，WeakHashMap特别适合做缓存。
