ConcurrentSkipListMap

发表于 2019-08-08 更新于 2019-08-10 分类于 Java集合

ConcurrentSkipListMap是基于跳表实现，跳表是一个随机化的数据结构，实质是一种可以进行二分查找的有序链表。跳表在原有的有序链表上面增加了多级索引，通过索引来实现快速查找。
调表不仅能提高搜索性能，同时也能提高插入和删除操作的性能。
